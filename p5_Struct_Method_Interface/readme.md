# 总结

## 结构体

## 方法
方法是绑定在结构体或结构体指针上的

## 接口
在 Go 语言中 interface resolution（接口实现） 是隐式的。如果传入的类型匹配接口需要的，则编译正确。

注意，在结构体指针上实现方法和结构体上实现方法有很大差异。在结构体上实现接口方法后，可直接使用结构体对象赋值给接口，而在结构体指针上实现接口方法后，需要使用`&结构体对象`才能赋值给接口。
